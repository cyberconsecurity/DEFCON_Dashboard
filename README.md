# DEFCON Dashboard

Full HACS-compatible Home Assistant integration providing:
- DEFCON Threat Level
- Functional Commands
- SPACECOM
- Flash alerts (10 min)
- Grafana export support
- Lovelace dashboards

## Installation
1. Copy `custom_components/defcon_dashboard` into your HA config
2. Restart Home Assistant
3. Add integration via UI
